<task type="auto">
  <name>Task 1: Download self-hosted fonts and wire @font-face declarations</name>
  <files>
    frontend/fonts/inter-latin-wght-normal.woff2
    frontend/fonts/jetbrains-mono-latin-wght-normal.woff2
    frontend/css/app.css
    frontend/index.html
  </files>
  <action>
1. Create `frontend/fonts/` directory.

2. Download the two variable woff2 font files:
   ```bash
   curl -L "https://cdn.jsdelivr.net/fontsource/fonts/inter:vf@latest/latin-wght-normal.woff2" \
        -o frontend/fonts/inter-latin-wght-normal.woff2
   curl -L "https://cdn.jsdelivr.net/fontsource/fonts/jetbrains-mono:vf@latest/latin-wght-normal.woff2" \
        -o frontend/fonts/jetbrains-mono-latin-wght-normal.woff2
   ```
   Verify each file is > 10KB (a valid woff2 is typically 50-100KB+). If curl fails or file is tiny, the CDN URL may have changed — try alternative Fontsource URLs.

3. Add `@font-face` declarations at the TOP of `frontend/css/app.css` (before `:root`):
   ```css
   @font-face {
     font-family: 'Inter Variable';
     font-style: normal;
     font-display: swap;
     font-weight: 100 900;
     src: url('../fonts/inter-latin-wght-normal.woff2') format('woff2-variations');
   }

   @font-face {
     font-family: 'JetBrains Mono Variable';
     font-style: normal;
     font-display: swap;
     font-weight: 100 800;
     src: url('../fonts/jetbrains-mono-latin-wght-normal.woff2') format('woff2-variations');
   }
   ```
   NOTE: The URL path `../fonts/` is relative to the CSS file at `frontend/css/app.css` — one level up, then into `fonts/`.

4. Update the `:root` font variables in `app.css`:
   ```css
   --font-d: 'Inter Variable', system-ui, sans-serif;   /* was: 'Syne', sans-serif */
   --font-m: 'JetBrains Mono Variable', monospace;      /* was: 'IBM Plex Mono', monospace */
   ```

5. Remove the three Google Fonts lines from `frontend/index.html` `<head>`:
   - `<link rel="preconnect" href="https://fonts.googleapis.com" />`
   - `<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />`
   - `<link href="https://fonts.googleapis.com/css2?family=Syne:wght@...&display=swap" rel="stylesheet" />`
   Leave the existing `<link rel="stylesheet" href="css/app.css" />` in place.
  </action>
